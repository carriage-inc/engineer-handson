## 要件

- トップ画面のユーザー登録ボタンをクリックすると、ユーザー登録画面に遷移するようにしてください。
- ユーザー登録画面の「登録」ボタンをクリックすると、ユーザー登録が完了し、リスト画面に遷移するようにしてください。
- ログイン画面でログインが成功すると、リスト画面に遷移するようにしてください。

## 提出方法

- 実装したファイルを GitHub にプッシュしてプルリクエストを作成し、 URL を共有してください。
