絵画専門の架空 EC サービス「イロアート」を、バックエンドとフロントエンドを分けて実装しましょう。  
バックエンドは RESTful API を Laravel で作成し、フロントエンドは NextJS で実装します。

デザインは以下に示す Figma ファイルを参考にしてください。
